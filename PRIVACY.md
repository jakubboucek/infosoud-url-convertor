# Privacy Policy / Zásady ochrany soukromí

## English
1. Data Collection

    The InfoSoud Fixer extension does not collect, store, or transmit any personal user data, identity information, or browsing history.

2. Data Processing (Redirection)

    The sole function of this extension is to detect outdated InfoSoud links (`justice.cz` domain). Due to technical limitations of Manifest V3, the conversion of old URLs to the new format is facilitated via a static web page `infosoud-fixer.web.app`.

    This page is hosted on the Firebase platform (Google).

    The URL conversion is performed using transparent JavaScript code running locally in the user's browser. 

    No URL parameters, court case details, or identifiable data are logged or stored on the server side during this process.

3. Third-Party Access

    No data is sold or shared with third parties. The project is open-source, allowing anyone to verify the data handling practices.

## Čeština
 
1. Sběr dat

    Rozšíření InfoSoud Fixer neshromažďuje, neukládá ani neodesílá žádné osobní údaje uživatelů, jejich identitu ani historii prohlížení.

2. Zpracování dat (Přesměrování)

    Jedinou funkcí rozšíření je detekce nefunkčních odkazů aplikace InfoSoud (doména `justice.cz`). Kvůli technickým omezením Manifestu V3 probíhá převod starého odkazu na nový formát prostřednictvím statické webové stránky `infosoud-fixer.web.app`.

    Tato stránka je hostována na platformě Firebase (Google).

    Převod URL probíhá pomocí transparentního JavaScript kódu přímo v prohlížeči uživatele.

    Během tohoto procesu nejsou na straně serveru ukládány žádné parametry URL, detaily soudních řízení ani data, která by mohla identifikovat uživatele.

3. Přístup třetích stran

    Žádná data nejsou prodávána ani sdílena se třetími stranami. Projekt je open-source, což umožňuje komukoliv ověřit způsob, jakým je s daty nakládáno.
